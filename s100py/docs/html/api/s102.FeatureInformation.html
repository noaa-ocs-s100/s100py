FeatureInformation¶

class s102.FeatureInformation(fill_empty=False)¶

Bases: HSTB.drivers.s100.s1xx.S1XX_Attributes_base

10.2.1 and table 10.2 and Table 10.1 of v2.0.0 This is used to describe the BathymetryCoverage and TrackingListCoverage within the GroupF feature listing. The features described under GroupF have a matching named entry parallel to GroupF (top level). The actual data (depths etc) is stored in the top level element while basic metadata is stored in this element.
