Attributes Documentation

scan_direction¶

The attribute scanDirection has the value class Sequence<CharacterString> a list of axis names that indicates the order in which grid points shall be mapped to position within the sequence of records of feature attribute values. The scan direction for all layers in S-102 is “Longitude” and “Latitude” or west to east, then south to north.

type¶

From S100 - CV_SequenceRule specified in ISO 19123. Only the values “linear” (for a simple regular cell size grid) and “Morton” (for a Quad Tree Grid) shall be used for data that conforms to this standard. While S102 further specifies - The default value is “linear”. No other options are allowed.

CodeList types are sets of strings (enumerations if all options are known). For SequenceType linear is lowercase while Morton is capitalized.
