scan_direction¶

The attribute scanDirection has the value class Sequence<CharacterString> a list of axis names that indicates the order in which grid points shall be mapped to position within the sequence of records of feature attribute values. The scan direction for all layers in S-102 is “Longitude” and “Latitude” or west to east, then south to north.

Return type

Union[ndarray, Dataset]
