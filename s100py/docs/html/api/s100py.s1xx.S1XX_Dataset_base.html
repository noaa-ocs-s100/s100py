S1XX_Dataset_base¶

class s100py.s1xx.S1XX_Dataset_base(*args, **opts)¶

Bases: list, s100py.s1xx.S1XX_WritesOwnGroup_base

The S102 spec stores some things as attributes that could (or should) be stored as attributes. This class reads/writes datasets but stores/accesses them as a list of class instances. Data access should then be used as object[index].attribute So for the FeatureInformation class that would be feat[0].name = “depth” and feat[1].name = “uncertainty”
