S102Root¶

class s100py.s102.s102.S102Root(recursively_create_children=False, **kywrds)¶

Bases: s100py.s100.S100Root

The root group contains a feature information group and N feature containers. In S102 there are currently two feature containers which are the ‘coverages’ bathymetry and tracking list. The coverage names are determined from the matching CoveragesAttributes 10.2 and Figure 10.1 of v2.0.0
